# Style Guide v1

## 한국어 문장부호
- 번역문에는 엠대시, 엔대시, 가로줄 계열 문장부호를 쓰지 않는다.
- 금지 문자: `—`, `–`, `―`, `─`, `--`.
- 위 금지 문자는 제목, 대사, 독백, 본문 어디에도 예외 없이 사용하지 않는다.
- 삽입구나 급격한 전환은 쉼표, 마침표, 괄호, 말줄임표 등 한국어 문장에 자연스러운 방식으로 풀어 쓴다.
- 장식용 구분선은 번역문 본문에 넣지 않는다.

## 인물 말투 일관성
- 한 작품 안에서 같은 인물의 말투, 높임법, 호칭은 일관되게 유지한다.
- 이전 화나 확정 예문에서 특정 인물이 하오체, 하십시오체, 해요체, 반말, 문어체 서술을 사용했다면 같은 관계와 상황에서는 그 패턴을 우선 따른다.
- 신분, 나이, 관계, 권력 차이에 맞는 말투를 유지하고 문장마다 임의로 바꾸지 않는다.
- 말투를 조정하더라도 의미, 감정 강도, 고유명사, 사건 정보는 바꾸지 않는다.
